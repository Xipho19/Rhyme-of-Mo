## 📖 项目简介

《墨韵物境》是一款以中国古代建筑与物理原理为主题的探索类小游戏合集。
玩家将在游戏中通过互动与解谜，理解古代建筑中的力学与结构智慧。

---

## ✨ 游戏特色

* 🧩 **拼图小游戏**：还原古代建筑结构
* 🏯 **图文匹配**：根据描述选择正确建筑
* ⚙️ **物理机制**：体验古代力学原理（如杠杆、结构支撑）
* 🎨 **中国风美术**：结合传统建筑与环境设计
