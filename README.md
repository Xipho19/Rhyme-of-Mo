## 📖 项目简介

《墨韵物境》是一款基于 RPG Maker MV 开发的教育类探索游戏。
玩家将在古代世界中，通过剧情互动与解谜玩法，理解《墨经》中的物理原理，实现“寓教于乐”的学习体验。

---

## 🧠 核心设计理念

* 🎓 教育性：将抽象物理知识转化为游戏机制（斜面、杠杆、重心等）
* 🎮 游戏性：探索 + 任务 + 战斗系统相结合
* 🏯 文化性：融合中国古代建筑与传统文化

---

## ✨ 主要玩法

### 🧩 物理解谜

* 推箱子（斜面原理）
* 机关互动（杠杆原理）
* 船只稳定（重心控制）

### 🗺 探索与任务系统

* 开放地图探索
* NPC 互动获取线索
* 多阶段任务推进剧情

### ⚔️ 战斗系统

* 回合制战斗机制
* 技能 / 魔法 / 道具系统
